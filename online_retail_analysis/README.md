# Анализ покупательского поведения интернет-магазина

## 📌 О проекте

Проект представляет собой комплексный анализ транзакционных данных интернет-магазина, специализирующегося на продаже подарков и товаров для дома. На основе данных о 541,910 транзакциях за период 2010-2011 годов проведена предобработка, исследовательский анализ, RFM-сегментация клиентов и анализ покупательской корзины.

**Основная цель:** выявить закономерности покупательского поведения, определить ценные сегменты клиентов и разработать рекомендации для повышения эффективности маркетинговых стратегий.

| Название столбца | Тип данных | Описание |
|-----------------|------------|----------|
| InvoiceNo | object | Номер заказа (префикс 'C' – возвраты) |
| StockCode | object | Уникальный код товара |
| Description | object | Наименование товара |
| Quantity | float | Количество товара (отрицательное – возвраты) |

| UnitPrice | float | Цена за единицу (£) |
| CustomerID | float | ID клиента (NaN – анонимные) |
| Country | object | Страна клиента |

Проект позволил выявить ключевые закономерности покупательского поведения, сегментировать клиентскую базу и предложить конкретные меры для увеличения среднего чека, удержания ценных клиентов и расширения международного присутствия.

Ожидаемые эффекты от внедрения:

↑ среднего чека на 15% (£26 → £30)

↑ доли международных продаж с 15.4% до 20%

↑ выручки на £104K за счёт возврата спящих гигантов
